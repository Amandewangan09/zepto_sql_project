# E-Commerce Startup Data Analytics Project

## Overview

This project focuses on analyzing real-world e-commerce startup data to uncover business insights and support data-driven decision-making. The analysis involved data cleaning, exploratory data analysis (EDA), and solving key business problems using SQL queries.

The goal was to transform raw transactional data into meaningful insights related to customer behavior, sales performance, product trends, and business growth opportunities.

---

## Business Objective

The e-commerce startup wanted to better understand its sales performance, customer purchasing patterns, and operational efficiency. This project addresses these objectives by analyzing historical transaction data and answering critical business questions through data analysis.

---

## Dataset

The dataset contains transactional records from an e-commerce startup, including:

* Customer information
* Order details
* Product categories
* Sales and revenue data
* Order dates and delivery information
* Payment and shipping details

**Dataset Type:** Real-world E-commerce Data

## Project Workflow

### 1. Data Cleaning

The raw dataset was cleaned to improve data quality and reliability.

Tasks performed:

* Handling missing values
* Removing duplicate records
* Correcting data types
* Standardizing column formats
* Identifying and treating inconsistencies

### 2. Exploratory Data Analysis (EDA)

EDA was performed to understand patterns, trends, and relationships within the data.

Key analyses included:

* Sales trend analysis
* Customer behavior analysis
* Product performance evaluation
* Revenue distribution
* Order volume trends
* Category-wise performance

Visualizations were created to communicate insights effectively.

### 3. SQL Business Analysis

SQL queries were used to solve real business problems and answer stakeholder questions.

Examples include:

* Top-selling products
* Highest revenue-generating categories
* Monthly sales performance
* Customer retention analysis
* Repeat customer identification
* Average order value calculation
* Regional sales comparison
* Revenue contribution by product category

---

## Key Business Questions Answered

1. Which products generate the highest revenue?
2. What are the top-performing product categories?
3. Which customers contribute the most sales?
4. How do sales vary across different months?
5. What is the average order value?
6. Which regions generate the highest revenue?
7. What percentage of customers are repeat buyers?
8. Which products should be prioritized for inventory planning?

## How to Run

### Clone the Repository

```bash
git clone https://github.com/yourusername/ecommerce-data-analytics.git
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Analysis

1. Open Jupyter Notebook.
2. Run the Data Cleaning notebook.
3. Run the EDA notebook.
4. Import the cleaned dataset into your SQL database.
5. Execute the SQL queries available in the `sql` folder.
